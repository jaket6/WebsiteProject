4x4x4 LED Cube
March 10, 2014


This particular project is not an original design but rather, a version of a project shown on the Instructables website (linked below).

With a no more than a spare Arduino-micro and a budget of about $20, I was able to recreate a 4x4x4 LED cube which ran through variety of lighting shows. Originally, the goal was to create a set of ten centerpieces for the IEEE End Of The Year Banquet. What better than a flashy, geometric, handmade LED display to show off what EE students can do? Well, as it turns out, something that required much less time and effort ended up being the wisest choice for a table centerpiece. Nonetheless, this weekend project turned out to be a rousing success.

Again, I am reusing pictures and material taken from the Instructable website to better illustrate this project!

Design


The design specified in this project is a simple one. The Arduino microcontroller has a combined 20 I/O pins. Sixteen of those outputs are used to control the voltage of each LED column while the remaining four outputs are used to control the horizontal ground planes. The LEDs are soldered layer by layer, with special attention given to the distance between each lead pin. The anode of each LED is connected to its respective column while the cathodes are attached to a common horizontal ground plane.


The ground planes are controlled through four 2n222 BJTs. Each BJT has a ground plane connected to the its collector while the circuit's common ground is soldered to the emitter. The base of the BJT is connected one of the Arduino’s I/O pins which will open/close each plane’s connection to ground (effectively completing the circuit).


The fully assembled cube can be controlled by alternating the Arduino outputs. Various Light shows can be custom made or download from an open-source library (links below). For this project, I used code which was made available by the author of the Instructables tutorial.


It was now time to make the fully soldered cube a little more presentable. A plastic box from the local craft store provided a fitting transparent case. A hole was drilled in the side of the case to provide access for the micro-USB cable. This allowed the pre-programmed cube to be plugged into any USB port and begin operation automatically. The last step was to hide the electronics located in the bottom of the case by painting the section IEEE blue. The picture below is the cube I managed to assemble.


Results



This turned out to be a pretty simple project and I encountered no major obstacles while putting it together (aside from all the soldering). The finished design looks great in a dark room and I would recommend this project for any beginner.
